Frequently Asked Questions

What does PageText do?

PageText converts any public web page into clean, readable Markdown format in one click. It is designed for feeding web content to LLMs like ChatGPT, Claude, and DeepSeek — no sign-up required, completely free.

Is PageText really free?

Yes. PageText is 100% free and open source. There are no usage limits, no API keys needed, and no registration required.

How does PageText protect my privacy?

All URL processing is client-side. Your URLs and converted Markdown never touch PageText servers. No personal data is collected or stored.

Can I use this for AI agents and LLM workflows?

Absolutely. The Markdown output is optimized for AI consumption. You can also call the same r.jina.ai endpoint directly from n8n, Zapier, or custom scripts.

What output formats are supported?

You can copy Markdown to clipboard or download as a .md file. The format is compatible with any text editor, Obsidian, Notion, or AI prompt interface.
